
# Backoffice

Microserviço com o CRUD dos usuários. 





## Deploy

Para rodar a aplicação você precisará do docker, docker-compose e nvm com node 16+.

```bash
git clone git@github.com:VFagionato/backoffice.git
```
```bash
npm install
```
```bash
docker-compose up
```
